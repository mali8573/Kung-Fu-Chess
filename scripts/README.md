שימוש

- ליצירת ZIP של כל קבצי המקור `.java` בפרויקט, הריצו את הסקריפט הבא מתוך שורת פקודה ב־Windows:

```
cd scripts
zip-java.bat
```

- ניתן להריץ ישירות את הסקריפט ב־PowerShell כדי לעבור פרמטרים מותאמים:

```
.\zip-java.ps1 -SourceDir ".." -OutDir "..\dist" -OutName "my-java.zip"
```

- הפלט יישמר בתיקיית `dist` ברמת השורש של הפרויקט. אם אין תיקייה כזו היא תיווצר אוטומטית.
